# Student Dashboard

This is a **Student Dashboard web application** developed as a **group project** using the MERN stack.  
The application helps students manage academic activities such as attendance, exams, quizzes, timetables, and daily productivity.

---

## Features

- Student login and signup
- Attendance calculator
- Exam countdown
- Timetable planner
- Quiz module
- Pomodoro focus timer
- Dashboard which has multiple student utilities

---

## Technologies Used

### Frontend
- React.js
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
